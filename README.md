# Loan Default Prediction Analysis

## Project Overview
This project aims to predict the probability of a client defaulting on a loan using Machine Learning. By analyzing historical financial behavior, the model identifies high-risk profiles to assist in credit decision-making.

## Tech Stack
* **Language:** Python 3.x
* **Libraries:** Pandas, NumPy, Scikit-Learn, Matplotlib, Seaborn.
* **Database:** SQL (for initial data extraction).

## Project Structure
- `notebooks/`: Exploratory Data Analysis (EDA) and Model Training.
- `data/`: Dataset storage (not tracked in Git).
- `models/`: Saved model weights.

## How to Run
1. Clone the repo: `git clone <your-url>`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the notebook in `notebooks/01_eda.ipynb`
